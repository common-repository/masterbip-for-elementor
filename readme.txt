== Description ==

Módulos (Widgets y addons) adicionales para Elementor.
Eres libre de modificar los estilos en tu hoja de estilo CSS según tus necesidades.

== Installation ==

1. Instalar de la forma tradicional.
2. Buscar los módulos en la barra elementor (casi siempre al final)
3. Seleccionar, arrastrar, configurar.

= Minimum Requirements =

* Elementor 2.0 or higher
